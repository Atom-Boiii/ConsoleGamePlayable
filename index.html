"Hacking" Game

This is a slightly larger project I made. I always wanted to know how a developer console was made. I wanted to make use of an argument system to make the console do diffrent things on different commands. I tried this system by making a game in which the player has to complete missions by hacking and breaking into places. This is a build with just a few features but it contains what I wanted.

Use email read (emailID) to read emails.
Email list to list all emails.
Email delete (emailID) to delete emails.
The home command sends you back to the console home.
